# PROJECT VELIR

> **An AI-First Historical Strategy Board Game Inspired by the Great Tamil Kingdoms**

---

## Overview

Project Velir is a turn-based historical strategy board game for Android that combines the accessibility of classic board games with strategic decision-making inspired by ancient Tamil history.

Players assume the role of one of the four great Tamil kingdoms:

* Chola
* Chera
* Pandya
* Pallava

Each kingdom competes for prestige, wealth, military dominance, diplomacy, and control over the mysterious ruins of **Velir Aram**, an ancient civilization believed to predate the Sangam Age.

Hidden beneath Velir Aram lies the forgotten **Ezhamandalam (The Seventh Realm)**, whose relics and secrets may decide the fate of Tamilakam.

---

# Vision

Create a board game that is easy to learn, difficult to master, and deeply replayable.

The game should combine:

* Strategic planning
* Resource management
* Kingdom politics
* Historical inspiration
* Exploration
* Risk vs Reward
* Emergent storytelling

Each match should generate memorable stories rather than repeating identical gameplay.

---

# Design Philosophy

Project Velir follows five core principles.

## 1. Simple Rules

Players should understand the core gameplay within five minutes.

---

## 2. Strategic Decisions

Every turn should present meaningful choices.

Players should constantly evaluate:

* Expand or defend?
* Trade or attack?
* Explore Velir Aram or strengthen the kingdom?
* Spend resources now or save them?

---

## 3. Historical Inspiration

The game is inspired by Tamil history, archaeology, trade, architecture, and Sangam-era kingdoms while remaining a fictional strategy experience.

This is not intended to be a historical simulation.

---

## 4. Mobile First

The game is designed specifically for portrait Android devices.

* Short sessions
* One-handed play
* Clean interface
* Fast loading
* Touch-first interactions

---

## 5. AI-First Development

Project Velir is developed using an AI-assisted workflow.

Every mechanic, document, asset specification, and gameplay system is documented before implementation.

Development follows modular engineering principles so both human developers and AI coding assistants can contribute consistently.
